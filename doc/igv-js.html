IGV JS

Introduction

IGV.js is an embeddable interactive genome visualization component written in JavaScript and CSS.
It is based on the Integrative Genomics Viewer(IGV), and developed by the same team.

Many genomic file formats are supported, including BAM, VCF, BED, and GFF, as well as the Global Alliance API.
